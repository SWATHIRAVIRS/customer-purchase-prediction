# Customer Purchase Prediction

## 📌 Project Overview

Customer Purchase Prediction is a Machine Learning project that predicts whether a customer will purchase a product based on different features such as age, salary, and other behavioral data.

This project demonstrates how machine learning models can help businesses understand customer behavior and make better marketing decisions.

---

## 🎯 Objective

The main objective of this project is to build a machine learning model that can predict customer purchase behavior using historical data.

Businesses can use this type of prediction to:

* Target the right customers
* Improve marketing strategies
* Increase sales and conversion rates

---

## 🗂 Dataset

The dataset contains customer information such as:

* Age
* Estimated Salary
* Gender
* Purchased (Target Variable)

The **Purchased** column indicates whether the customer bought the product or not.

## ⚙️ Machine Learning Model

The following steps were used to build the model:

1. Data Collection
2. Data Preprocessing
3. Feature Selection

The model learns patterns from the dataset and predicts whether a new customer is likely to purchase the product.
